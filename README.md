# 📊 Sales Performance Dashboard – Power BI

## 📌 Overview
This project analyzes **₹769M+ sales** across **4K transactions** using Power BI.  
Built an interactive dashboard using **DAX, data modeling, and relationships** to uncover key business insights.

---

## 🚀 Key Insights
- Top-selling brands contributed **60%+** of total sales  
- Automated reporting reduced manual effort by **50%**  
- Identified revenue patterns, payment trends, and seasonal behavior  
- Improved stakeholder decision-making speed by **35%**
